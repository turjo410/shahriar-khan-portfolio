# English Proficiency Documents

Place your English proficiency test reports here:

## Expected Files:
- `TOEFL_Report.pdf` - Your TOEFL iBT score report

## Current Configuration:
The credentials page expects the TOEFL report at:
- `/documents/english-proficiency/TOEFL_Report.pdf`

## To add your TOEFL report:
1. Save your official TOEFL score report as PDF
2. Rename it to `TOEFL_Report.pdf`
3. Place it in this folder

The score displayed on the website (93/120) matches the data in the credentials page.
